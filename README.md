# IMMPACT Employer Dashboard (React JSX + Tailwind v4)

Standalone React project — no TypeScript, no TanStack. Uses `react-router-dom` and Tailwind CSS v4 via the Vite plugin.

## Run

```bash
npm install
npm run dev
```

Open http://localhost:5173 — the app redirects to `/employer`.

## Pages

- `/employer` — Overview dashboard
- `/employer/jobs` — Job listings with status filters
- `/employer/post-job` — 4-step job posting wizard with live preview
- `/employer/candidates` — Kanban / list view of applicants
- `/employer/interviews` — Interview schedule
- `/employer/messages` — Threaded messaging
- `/employer/company` — Company profile & sponsor license
- `/employer/profile` — Personal profile & security

All styling uses semantic tokens defined in `src/styles.css`.
